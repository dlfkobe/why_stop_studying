## 债券
- ### 实体类型

| 实体类型     | 中文含义 | 实体数量   |   举例    |
| :---        |    :----:   |          ---: |     ---:   |
| ts_code     | 转债代码     |   |       |
| stk_code   |  正股代码     |     |       |      


- ### 实体关系类型

| 实体关系类型    | 中文含义 | 关系数量   |   举例    |
| :---        |    :----:   |          ---: |     ---:   |
|   match_to   | 对应正股是      |   |    <125002.SZ,对应正股是,000002.SZ>   |


-  ### 属性类型

以下都为转债代码的属性，正股代码属性与其重复，因此不再单独设置
| 属性类型    | 中文含义 |   举例    |
| :---        |    :----:   |     ---:   |
| list_date      |    上市日期    |       |
| par  | 面值    |       |
| issue_size  | 发行总额（元）    |       |
| bond_full_name  | 转债名称    |       |
| bond_short_name  | 转债简称    |       |
| pay_per_year  | 年付息次数   |       |
| value_date  | 起息日期   |       |
| maturity_date  | 到期日期   |       |
| exchange  | 上市地点  |       |

- ### 实体类型

| 实体类型     | 中文含义 | 实体数量   |   举例    |
| :---        |    :----:   |          ---: |     ---:   |
| ts_code     | 基金代码     |   |       |
| manager_name   |  基金经理（名字）    |     |       |      
| management  |  管理人    |     |       |     
 


- ### 实体关系类型

| 实体关系类型    | 中文含义 | 关系数量   |   举例    |
| :---        |    :----:   |          ---: |     ---:   |
|  manage_by_who    | 基金经理是(被谁管理)     |   |    <150018.SZ,经理是,周毅>   |

|  served_at    | 基金经理任职于     |   |    <周毅,任职于,银华基金>   |
